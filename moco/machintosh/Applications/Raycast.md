# [MacApps] Raycast

[Raycast](https://www.raycast.com/) 是一款強大的 macOS 生產力工具，可以取代 Spotlight 和 Alfred，提供更快速、更強大的應用程式啟動和工作流程自動化功能。

## 安裝

```shell
brew install raycast
```

## 主要功能

### 快速啟動

- 應用程式快速啟動
- 檔案搜尋
- 計算機功能
- 系統指令執行

### 擴充功能

Raycast 擁有豐富的擴充功能生態系統：

- **剪貼簿歷史記錄** - 管理複製歷史
- **視窗管理** - 快速調整視窗大小和位置
- **翻譯工具** - 即時翻譯
- **GitHub 整合** - 快速存取 GitHub 資源
- **Jira/Linear** - 專案管理整合
- **自訂腳本** - 支援自訂 Script Commands

### Script Commands

支援使用 Shell、Python、Ruby、Node.js 等語言撰寫自訂指令，快速執行常用任務。

## 優勢

- **免費** - 核心功能完全免費
- **快速** - 啟動速度極快
- **美觀** - 現代化的 UI 設計
- **擴充性強** - 豐富的擴充功能商店
- **活躍社群** - 持續更新和改進
