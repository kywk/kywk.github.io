# [Obs] 快速鍵

## Default 

| 功能      |  快速鍵  | 備註  |
| ------- | :---: | --- |
| 打开控制面板  | `⌘ P` |     |
| 打开快速启动器 | `⌘ O` |     |

## My Setting

| 功能           | 快速建   |
| ------------ | ----- |
| 打開左側 Sidebar | `⌥ ←` |
| 打開右側 Sidebar | `⌥ →` |

### Key Map
