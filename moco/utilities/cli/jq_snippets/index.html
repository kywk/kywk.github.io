Trick & Tips​

  • 在 BASH script 使用 jq 時, 若是透過 VAR=jq '.["foo"]' 來設定變數為 jq parse 結果, 字串值 "example" 是代表 '"example"' 而非 'example'
    • 承上, 若 jq 查不到資料的 null 實際上代表 'null'
    • 這是由於 BASH script 是利用一個字串去接收 jq 查詢結果的 stdout 所導致
