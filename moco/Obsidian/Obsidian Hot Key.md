# [Obsidian] Hot Keys - 快速鍵大全

快速鍵是提升 Obsidian 使用效率的關鍵。熟練掌握快速鍵可以大幅減少滑鼠操作，提升筆記和知識管理的效率。

## 預設快速鍵

### 📝 基本操作

| 功能 | macOS | Windows/Linux | 說明 |
|------|-------|---------------|------|
| 開啟命令面板 | `⌘ P` | `Ctrl P` | 快速執行命令 |
| 快速切換 | `⌘ O` | `Ctrl O` | 快速開啟文件 |
| 全域搜尋 | `⌘ ⇧ F` | `Ctrl Shift F` | 搜尋所有文件 |
| 新建筆記 | `⌘ N` | `Ctrl N` | 建立新文件 |
| 開啟設定 | `⌘ ,` | `Ctrl ,` | 開啟設定頁面 |

### 📊 編輯功能

| 功能 | macOS | Windows/Linux | 說明 |
|------|-------|---------------|------|
| 粗體 | `⌘ B` | `Ctrl B` | 加粗選中文字 |
| 斜體 | `⌘ I` | `Ctrl I` | 斜體選中文字 |
| 內嵌程式碼 | `⌘ E` | `Ctrl E` | 內嵌程式碼格式 |
| 外部連結 | `⌘ K` | `Ctrl K` | 建立外部連結 |
| 縮排 | `⌘ ]` | `Ctrl ]` | 增加縮排 |
| 取消縮排 | `⌘ [` | `Ctrl [` | 減少縮排 |

### 🔍 導航功能

| 功能 | macOS | Windows/Linux | 說明 |
|------|-------|---------------|------|
| 後退 | `⌘ ⌥ ←` | `Ctrl Alt ←` | 回到上一個位置 |
| 前進 | `⌘ ⌥ →` | `Ctrl Alt →` | 前進到下一個位置 |
| 跳轉到行 | `⌘ G` | `Ctrl G` | 跳轉到指定行號 |
| 檔案總管 | `⌘ ⇧ E` | `Ctrl Shift E` | 開啟/關閉檔案總管 |

### 📊 視圖控制

| 功能 | macOS | Windows/Linux | 說明 |
|------|-------|---------------|------|
| 切換編輯/預覽 | `⌘ E` | `Ctrl E` | 在編輯和預覽模式間切換 |
| 關係圖譜 | `⌘ G` | `Ctrl G` | 開啟關係圖譜 |
| 左側邊欄 | `⌘ ⇧ L` | `Ctrl Shift L` | 開啟/關閉左側邊欄 |
| 右側邊欄 | `⌘ ⇧ R` | `Ctrl Shift R` | 開啟/關閉右側邊欄 |

### 🔧 設定方法
1. 開啟設定 (`⌘ ,`)
2. 點擊「快速鍵」分頁
3. 搜尋要設定的功能
4. 點擊「+」按鈕新增快速鍵
5. 按下想要的組合鍵

## 進階技巧

### 🔄 工作流程快速鍵

#### 日常筆記流程
```
1. ⌘ ⇧ D     # 開啟今日日記
2. ⌘ ⇧ I     # 快速記錄想法
3. ⌘ P         # 命令面板執行功能
4. ⌥ ←       # 開啟左側邊欄瀏覽
5. ⌥ →       # 開啟右側邊欄查看關聯
```

#### 研究寫作流程
```
1. ⌘ O         # 快速開啟相關文件
2. ⌘ ⇧ F     # 搜尋相關內容
3. ⌘ K         # 建立引用連結
4. ⌘ E         # 切換預覽模式檢查
5. ⌘ ⇧ E     # 開啟檔案總管整理
```

### 📊 組合鍵技巧

#### 多游標編輯
- `⌥ + 點擊`: 新增游標
- `⌘ D`: 選擇相同內容
- `⌘ ⇧ L`: 選擇整行

#### 快速導航
- `⌘ ⇧ O`: 在新分頁開啟文件
- `⌘ W`: 關閉當前分頁
- `⌘ ⇧ T`: 重新開啟最近關閉的分頁

### 🎨 自訂工作區

#### 建立工作區快速鍵
```
⌘ 1: 切換到工作區 1 (寫作模式)
⌘ 2: 切換到工作區 2 (閱讀模式)
⌘ 3: 切換到工作區 3 (管理模式)
```

## 快速鍵記憶法

### 🧠 記憶技巧

#### 邏輯分組
- **文件操作**: `⌘ + 字母` (新增、開啟、儲存)
- **編輯功能**: `⌘ + 字母` (粗體、斜體、連結)
- **視圖控制**: `⌘ ⇧ + 字母` (邊欄、面板)
- **插件功能**: `⌘ ⇧ + 字母` (模板、任務)

#### 常用組合記憶
- `⌘ P`: **P**alette (命令面板)
- `⌘ O`: **O**pen (開啟文件)
- `⌘ N`: **N**ew (新建文件)
- `⌘ K`: **K**ink (建立連結)
- `⌘ E`: **E**dit/Preview (編輯模式)

### 📝 練習建議
1. **每日練習**: 選擇 3-5 個常用快速鍵每日練習
2. **逐步增加**: 熟練一組後再學習下一組
3. **建立習慣**: 強迫自己使用快速鍵而非滑鼠
4. **定期複習**: 每週複習一次所有快速鍵

- **切換文件**: 滑鼠 2 秒 vs 快速鍵 0.3 秒
- **格式化文字**: 滑鼠 4 秒 vs 快速鍵 0.2 秒

## 常見問題

### Q: 快速鍵衝突怎麼辦？
A: 在設定中搜尋衝突的快速鍵，重新分配或刪除不需要的。

### Q: 如何備份快速鍵設定？
A: 快速鍵設定儲存在 `.obsidian/hotkeys.json`，可以複製到其他裝置。

### Q: 插件快速鍵不生效？
A: 確認插件已啟用，並檢查是否與其他快速鍵衝突。
